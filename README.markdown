Twisted based, pluggable IRC bot
--------------------------------


Required:
=========

 * Python >= 2.6
 * [twisted](http://twistedmatrix.com/) library
 * [zope-interface](http://pypi.python.org/pypi/zope.interface)


Optional dependencies:
======================

 * feedreader plugin:
    * [python-feedparser](http://www.feedparser.org/)
