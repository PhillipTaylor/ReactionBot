Twisted based, pluggable IRC bot
--------------------------------


Required:
=========

 * Python >= 2.6
 * [twisted](http://twistedmatrix.com/) library
 * [twisted-words](http://twistedmatrix.com/trac/wiki/TwistedWords)


Optional dependencies:
======================

 * feedreader plugin:
    * [python-feedparser](http://www.feedparser.org/)
